# Data Analysis

This folder contains the playbooks, checklists, and references for performing rigorous data analysis on hardware experiment results and other datasets.

## Pillars

- **`skills.md`**: The primary skill definition covering statistical methods, machine learning guards, and notebook patterns.
- **`ideas/`**: Backlog of analysis techniques or automation scripts to implement.

## Usage

When starting a new analysis (e.g. in a Jupyter notebook), refer to `skills.md` for the mandatory **Analysis Checklist** and **Notebook Structure Pattern**.
