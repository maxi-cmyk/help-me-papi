# Tools

Top-level area for automating Developer Experience (DevEx) via scripts, Command Line Interfaces (CLIs), and Model Context Protocol (MCP) servers.

## Pillars

- **`skills/`**: Foundational playbooks on how to build and design our internal CLIs and MCP servers.
- **`data-analysis/`**: Playbooks for statistical testing, machine learning guards, and hardware experiment analysis.
- **`ideas/`**: Backlog of repetitive tasks that should be automated eventually.
- **`guidelines.md`**: Consolidated standards on what stack to use for automation (e.g. `argparse`, Node.js) and what off-the-shelf tools to install.
- **`PROMPTS.md`**: Prompt templates to pass into AI so it can scaffold tools for you instantly.
