# Security Resources

## OWASP references
- https://owasp.org/www-project-top-ten — OWASP Top 10 (the canonical list of critical web security risks)
- https://cheatsheetseries.owasp.org/cheatsheets/Injection_Prevention_Cheat_Sheet.html — injection prevention
- https://cheatsheetseries.owasp.org/cheatsheets/Input_Validation_Cheat_Sheet.html — input validation
- https://cheatsheetseries.owasp.org/cheatsheets/Cross-Site_Request_Forgery_Prevention_Cheat_Sheet.html — CSRF
- https://cheatsheetseries.owasp.org/cheatsheets/Cross_Site_Scripting_Prevention_Cheat_Sheet.html — XSS
- https://cheatsheetseries.owasp.org/cheatsheets/Password_Storage_Cheat_Sheet.html — password hashing

## Libraries
- https://helmetjs.github.io — Helmet (security headers for Express)
- https://zod.dev — Zod (input validation)
- https://github.com/express-rate-limit/express-rate-limit — express-rate-limit
- https://github.com/express-rate-limit/express-slow-down — express-slow-down
- https://github.com/ranisalt/node-argon2 — argon2 for Node.js
- https://github.com/nicolo-ribaudo/isomorphic-dompurify — DOMPurify for HTML sanitization

## Security scanning
- https://github.com/gitleaks/gitleaks — scan git history for accidentally committed secrets
- https://www.npmjs.com/package/snyk — Snyk (dependency vulnerability scanning, integrates with CI)
- https://securityheaders.com — check your deployed site's security headers

## HTTP security headers reference
- https://developer.mozilla.org/en-US/docs/Web/HTTP/Headers/Content-Security-Policy — CSP reference
- https://developer.mozilla.org/en-US/docs/Web/HTTP/Headers/Strict-Transport-Security — HSTS
- https://developer.mozilla.org/en-US/docs/Web/HTTP/Cookies#security — cookie security flags
