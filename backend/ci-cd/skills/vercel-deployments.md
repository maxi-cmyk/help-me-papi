# Vercel Deployments

Rules and configuration for Vercel.

## Workflow
- Vercel automatically deploys every branch (Preview) and the main branch (Production).
- Connect via GitHub integration for zero-downtime automatic builds.
- Configure Environment Variables via the Vercel Dashboard for Production and Preview separately/securely.
