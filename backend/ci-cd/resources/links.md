# CI/CD Resources

## Vercel
- https://vercel.com/docs/deployments/overview — how Vercel deployments work
- https://vercel.com/docs/projects/environment-variables — env vars per environment
- https://vercel.com/docs/deployments/preview-deployments — preview URLs on every branch
- https://vercel.com/docs/monorepos — monorepo setup if frontend/ and backend/ are separate

## GitHub Actions
- https://docs.github.com/en/actions/quickstart — quickstart
- https://docs.github.com/en/actions/using-workflows/workflow-syntax-for-github-actions — full YAML syntax reference
- https://github.com/marketplace/actions — marketplace of prebuilt actions (setup-node, checkout, etc.)
- https://docs.github.com/en/actions/security-guides/encrypted-secrets — storing secrets securely

- https://jestjs.io — the classic, widely used
- https://testing-library.com — for testing React components

- https://www.typescriptlang.org/tsconfig — tsconfig reference for `tsc --noEmit`
