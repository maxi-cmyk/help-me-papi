# Database Resources

## PostgreSQL
- https://www.postgresql.org/docs/current/indexes.html — indexes guide
- https://explain.depesz.com — paste EXPLAIN ANALYZE output, get a visual breakdown
- https://pganalyze.com/docs — slow query analysis
- https://supabase.com/docs — Postgres as a service with auth, realtime, and storage
- https://neon.tech/docs — serverless Postgres (great for preview envs per branch)

## Redis
- https://redis.io/docs/data-types — data types (strings, sets, sorted sets, hashes)
- https://upstash.com/docs/redis/overall/getstarted — Upstash serverless Redis

## ClickHouse
- https://clickhouse.com/docs/en/intro — ClickHouse intro
- https://clickhouse.com/docs/en/getting-started/quick-start — quick start
- https://tinybird.co/docs — ClickHouse with an API layer, startup-friendly
- https://clickhouse.com/blog/columnar-vs-row-oriented — columnar vs row storage explained

## ORMs and query builders
- https://www.prisma.io/docs — Prisma (TypeScript ORM, great DX)
- https://orm.drizzle.team — Drizzle (lightweight, SQL-like, TypeScript)
- https://knexjs.org — Knex (query builder, no ORM opinions)

## Schema migrations
- https://www.prisma.io/docs/orm/prisma-migrate — Prisma Migrate
- https://orm.drizzle.team/docs/migrations — Drizzle migrations

## Search
- https://typesense.org/docs — Typesense (easier than Elasticsearch)
- https://www.elastic.co/guide/en/elasticsearch/reference/current/index.html — Elasticsearch

## General
- https://use-the-index-luke.com — free book on SQL indexing, extremely practical
- https://architecturenotes.co/things-you-should-know-about-databases — good mental models overview
