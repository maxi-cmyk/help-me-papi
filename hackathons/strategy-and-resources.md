# Hackathon Strategy & Resources

References and tools that help evaluate sponsors, themes, and winning criteria.

---

### Pre-Hackathon Checklist
- [ ] Create GitHub repo and invite all team members.
- [ ] Provision initial backend services (e.g., Supabase project, Vercel deployment link).
- [ ] Agree on the tech stack, component library, and aesthetic before writing code (`Next.js + React` for web, `Next.js + Electron` for desktop apps).

## Part 1: Strategic Playbook

- **Evaluate Landscape:** Quickly identify sponsor goals and judge preferences.
- **Research Research Research** Understand target user and sponsor's pain points, how to improve user flow and reduce friction inusage/understanding. 
- **Understand User Personas:** Map out realistic personas to solve concrete problems after adequate researtch is complete.
- **Low-Fidelity Prototyping:** Immediately after persona generation, start low-fidelity prototyping in parallel with early dev setup. Use **Figma**, **Stitch**, or even **hand-drawing** to map out screens and interactions. This ensures the team is aligned on the UX before heavy coding begins.
- **Focus on Flow:** Build an end-to-end user flow that can be demoed cleanly in 3 minutes.
- **Fake the Unnecessary:** Hardcode non-essential data if it doesn't affect the core value proposition.

## Part 2: References & Tools

### Tools & Stacks
- **Web App Boilerplate:** React + Next.js
- **Desktop App Boilerplate:** Electron + Next.js
- **Demo Recording & Editing:** CapCut or iMovie.
- **Pitch Scripting:** Write the core script yourself based on the persona, then use AI to refine it. **Crucial:** Always inject a touch of humor and emotional emphasis to genuinely connect with the target audience and stand out from dry technical pitches.

### References & Pitch Inspiration
- **The "Story" Structure:** Hook (Persona's pain point) -> Solution (Your app) -> Demo (70% of the time, the AHA moment) -> Tech Stack -> Vision/Impact.
- **Classic MVP Demo:** Study the original [Dropbox demo video](https://techcrunch.com/2011/10/19/dropbox-minimal-viable-product/) for how to pitch a complex product in a dead-simple, relatable way.
- **Airbnb Pitch Deck:** The gold standard for keeping slides minimal and focusing intensely on the problem/solution metric.
