# AI

Top-level area for AI-related skills, resources, prompts, and ideas.

## Subdomains

- **RAG** (Retrieval-Augmented Generation)
- **ML** (Machine Learning workflows)

Each subdomain is structured into core pillars to prevent context fragmentation:

- **`skills/`**: Foundational playbooks, tools, and technical capabilities.
- **`ideas/`**: Scrappy thoughts, feature notes, and experiments.
- **`guidelines.md`**: Consolidated standards, decisions, toolings, and references.
- **`PROMPTS.md`**: Copy-paste prompt templates for planning, implementing, and evaluating AI work.
